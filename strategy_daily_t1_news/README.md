# 1月频扩张前向验证 (Expanding WFO) 新闻 T+1 策略生产环境

该项目是基于 A 股新闻舆情（THS 个股新闻）、同花顺热度、筹码分布及市值中军偏好构建的严格 T+1 日线交易系统。

## 核心目录结构
- `daily_prediction.py`: **每日生产脚本**。收盘后运行，自动更新模型并生成次日 Top 3 观察池。
- `train_model.py`: 核心模型训练引擎（XGBoost），支持无限扩张记忆训练。
- `backtest_simulation.py`: WFO 动态回测框架。用于复盘或调整参数。
- `mktcap_sensitivity.py`: 市值敏感度实验脚本。

## 生产运行流程 (Daily Workflow)

### 1. 数据对齐 (Data Sync)
确保 `D:\iquant_data\data_v2` 目录下的 Parquet 数据已更新至当日。
主要依赖：`data_day1` (价格), `ths_rank1` (热度), `cyq1` (筹码), `other_day1` (市值), `news_data` (舆情)。

### 2. 执行预测 (Get Signals)
- **方法 A: 全量自动预测** (含数据同步、模型更新):
```bash
python daily_prediction.py
```
- **方法 B: 一键指定日期预测** (实战推荐，不重训模型，同步指定日数据):
```bash
# 赋予执行权限 (Windows Git Bash 环境)
chmod +x one_click_predict.sh
# 执行预测今日 (或指定日期)
./one_click_predict.sh 20260325
```
脚本会自动：
1. 调用 `data_fetcher.py` 补全 Tushare 数据。
2. 调用 `production_predict.py` 使用预存模型生成名单。
3. 应用 500 亿市值上限 & 剔除 688 逻辑。
4. 打印次日开盘建议买入的 3 只标的。

### 3. 持仓管理 (T+1 Execution)
- **买入**：次日 9:30 以**开盘价**买入建议标的。若开盘涨停（主板>9.5% / 创业板>19.5%）则放弃该股。
- **卖出**：
    - 盘中触及买入价 +4% 自动止盈（实现日内高点锁定）。
    - 若全天未触发止盈，则在**当日收盘前**（14:55~15:00）全仓卖出（严格 T+1）。

## 策略核心逻辑总结
- **模型**：XGBoost 分类器，预测次日高点 >4% 的概率。
- **特征**：热度分位数、筹码集中度、获利盘比例、市场/个股舆情得分。
- **宇宙**：主板 + 创业板（不含 688），流通市值限制在 500 亿以下（锁定中军龙头）。
- **年化预期**：在 2024-2026 回测中，该组合配置录得约 **194.46%** 的复利年化收益，夏普比率 **3.37**。
