## 一、系统架构总览

`quant_system_v2` 是一个围绕 **A股T+1市场** 构建的多策略量化研究平台，包含以下并行研究线：

```
quant_system_v2/
├── Study 005（核心短周期策略）          ← 期权增强XGBoost双模型，T+1/T+2持仓
├── Study 004（系统化研究框架）           ← 54特征全量实验，已归档教训
├── longterm-research（长期多因子）        ← 20日再平衡，Ridge排序，100只持仓
├── smart-beta-research（Smart Beta）     ← 小盘底仓+VIX择时
├── option-research（期权学术研究）        ← Max Pain引力假说，严格计量校正
├── final_method（Doubao选股）            ← 新闻+筹码，概率阈值驱动
├── new_idea（特征创新）                   ← 筹码Delta特征，增量信息显著
├── 实盘部署层                             ← API Server + PTrade + 晨报邮件
```

## 二、各研究线核心成果与教训

### 2.1 Study 005 — 期权增强型全局策略（当前主推）

**策略定位**：短周期（T+1买入→T+2卖出）个股Alpha策略，结合大盘期权情绪作为"安全闸"。

**核心架构**：
- **XGBoost 双模型**：Model 1 预测 `>=+6%` 上涨概率，Model 2 预测 `<=-5%` 暴跌概率
- **Walk-Forward 月滚动重训**：3年训练窗口，每月初自动追加数据重训
- **特征三维度**：个股技术指标 + 盘前新闻NLP（GLM-4）+ 大盘期权（QVIX/PCR）

**交易规则（物理级去未来函数）**：
- T日收盘预测 → T+1日 9:30 以**开盘价**买入
- T+1日强制锁仓（T+1制度）
- T+2日：
  - 盘中触及+6% → 止盈
  - 盘中跌破-5% → 止损（Worst-Case：先止损）
  - 14:50 未触发 → 收盘价强制平仓
  - 一字跌停 → 顺延至下一交易日

**回测绩效（2022-2026，严格去未来函数）**：

| 指标 | Baseline（无期权） | 期权增强 | 变化 |
|------|------------------|----------|------|
| 全周期累计收益 | -13.5% | -10.5% | +3.0% |
| 全周期年化 | -3.5% | -2.7% | +0.8% |
| 全周期Sharpe | -0.68 | -0.70 | 接近 |
| 全周期最大回撤 | -17.4% | -13.0% | **+4.4%（回撤减小）** |
| 样本外(2025-2026)年化 | -5.2% | -1.5% | **+3.7%** |
| 样本外Sharpe | -1.19 | -0.37 | **+0.82** |
| 样本外最大回撤 | -9.2% | -4.0% | **+5.2%** |

**关键教训**：
1. 策略在A股T+1锁仓约束下**绝对收益为负**，但期权特征显著**减震抗跌**
2. 硬性期权开关（QVIX高时暂停交易）导致踏空，**模型级特征堆叠**远优于宏观开关
3. 训练标签过度拟合高波动个股，需重新审视标签设计

**实盘部署**：
- `daily_morning_pipeline.py`：每日8:00自动运行，推送HTML晨报邮件
- `ptrade_client_v5.py`：PTrade恒生柜台全自动交易（含盘前对账、ST过滤、流动性ADV 1%限制、本地Parquet容灾）
- `api_server.py`：FastAPI本地服务，含板块超限过滤（单行业≤2只）+ 双模型熔断（prob_crash≤15%）
- 每日最大持仓3只，每只1/6总仓位，留1/2现金缓冲

### 2.2 Study 004 — 系统化研究框架（教训档案）

**策略定位**：1日/5日/28日持仓目标的全量特征实验（54特征）。

**关键发现**：
1. **数据泄漏是最大杀手**：v2的Validation CAGR 160%是虚假的，Walk-Forward后真实CAGR仅22%
2. **T+1约束下止损完全失效**：无T+1时-3%止损让CAGR从1.6%暴涨到248%，但加入T+1约束后所有止损组合都巨亏（-62%~-98%）
3. **跳空低开绕过止损**：A股买入当天无法卖出，次日开盘可能已远低于止损价
4. **止盈5%成为T+1下唯一可行参数**：无止损+止盈5%是T+1约束下唯一正收益组合（opt CAGR 41.3%, test CAGR +0.7%）
5. **原始策略没有正期望**：无止损+无止盈=必亏，说明XGBoost分类器对1日收益预测能力不足

**核心教训**：**任何不考虑T+1约束的回测都是虚假回测。**

### 2.3 Long-Term Research — 长期多因子排名策略（最成熟的策略）

**策略定位**：20日再平衡，Walk-Forward月度Ridge回归排序，持仓100只，追求统计显著Alpha。

**核心成果**：
1. **去污染后IC反而提升**：移除`ths_hot`短样本因子后，Daily Rank IC从+0.0987提升到+0.1002
2. **全周期表现（2022-2026，OOS）**：
   - 纯多因子：+59.94%总收益，12.42% CAGR，0.73 Sharpe，-28.03% MaxDD
   - 期权风控版：+46.89%总收益，10.06% CAGR，0.65 Sharpe，-28.27% MaxDD
   - 基准CSI 1000：+4.77%总收益，1.17% CAGR，0.17 Sharpe，-46.22% MaxDD
   - 市场等权：+46.98%总收益，10.08% CAGR，0.54 Sharpe，-29.97% MaxDD
3. **风格归因确认真实Alpha**：
   - 控制市场+Size+行业后，年化Alpha = +11.90%（t-stat 2.36，p=0.018，**统计显著**）
   - 但策略收益73%-79%由风格暴露（主要是小盘SMB）解释
   - 策略相对等权基准的纯选股Alpha约13%

**关键教训**：
1. **小盘Beta是主要收益来源**：策略大幅跑赢CSI 1000主要是等权小盘暴露，而非纯选股
2. **Alpha是周期性的**：2023(+17.34%)、2025(+20.99%)大幅跑赢，但2024(-1.53%)、2026(-5.56%)跑输
3. **盲测期Alpha不显著**：2025-09至2026-03盲测期，R²=98%，Alpha不显著（t=0.64），说明近期风格主导
4. **新闻因子几乎零增量**：Config B(+News) vs A(Baseline)：Alpha仅+0.08%，Sharpe从2.31→2.32
5. **THS热排因子是噪音**：加入`ths_hot_score`后收益从95.35%降至87.76%（-7.59%），应排除
6. **Alpha101预处理是关键**：原始Alpha101直接输入会毒化模型（Sharpe 0.73→0.43），必须经Winsorize+行业中性化+Z-Score标准化

**未来方向**：
- PCA/LASSO正交化解决96特征多重共线性
- 从Ridge切换到XGBoost提取非线性交互

### 2.4 Smart-Beta Research — 小盘Smart Beta底仓

**策略定位**：接受"小盘Beta"现实，用多因子过滤垃圾股，构建低波动小盘替代指数。

**核心成果**：
- 无风控版：88.23%总收益，54.75% CAGR，2.02 Sharpe，-17.21% MaxDD
- 趋势+VIX风控版：39.94%总收益，26.11% CAGR，1.54 Sharpe，-15.57% MaxDD
- 完美跟踪CSI 1000（84.52%），但波动更低（23.05% vs 27.51%），Sharpe更高（2.02 vs 1.68）

**关键教训**：
- 在暴力牛市中，MA20趋势滞后导致踏空（V型反弹错过）
- 纯分散版是优秀的"底仓"，风控版是"睡得着觉"的保守选择

### 2.5 Option Research — 期权Max Pain学术研究

**策略定位**：对50ETF/300ETF期权"最大痛点"效应进行严格学术级计量检验。

**核心成果**：
1. **前视偏差修正**：使用T-1日持仓数据计算T日Max Pain，消除未来函数
2. **Placebo检验通过**：替换为"最近行权价"后不显著，证明Max Pain效应是特定因果
3. **反转效应控制**：加入Ret5d控制后，临期DTE<=3的Max Pain效应从显著变为**不显著**
4. **⚠️ 重大发现**：2025-2026样本外，**全周期All DTE窗口出现极强显著性**（50ETF t=-3.42，300ETF t=-2.91）
   - 解释：A股期权市场2025年后发生**结构性转变**，做市商Delta对冲产生月度均值回归拉力
5. **实际可交易夏普接近零**：多空策略Sharpe 0.04/-0.04，无法独立盈利

**应用价值**：可作为**月度组合权重偏置器**或**指数持仓Alpha增强因子**

### 2.6 Final Method — Doubao选股策略（早期实验）

**策略定位**：基于盘前新闻NLP + 筹码分布的T+1/T+2策略。

**核心发现**：
- prob>0.8时胜率100%（7笔），0.7-0.8时胜率92.3%（6笔）
- **prob<0.6时期望为负，必须空仓**
- 核心纪律：宁可空仓，不推亏钱的票

**模型**：仅5个特征（hot_rank_pct, chip_concentration, winner_rate, news_market_impact, news_stock_impact）

### 2.7 New Idea — 筹码Delta特征创新

**核心发现**：
- 新增**筹码Delta特征**（t0-t-1的筹码结构变化）贡献巨大：
  - `delta_cost_5pct` 重要性第1（0.143）
  - `delta_cost_15pct` 重要性第2（0.133）
  - 原有特征重要性普遍下降，说明新增特征提供**增量信息**
- **结论**：筹码结构的变化是预测次日走势的重要信号

## 四、量化核心教训总结

### 教训1：A股T+1是量化策略的"物理定律"
- 买入当天无法卖出，任何"盘中止损"回测都是虚假回测
- 次日跳空低开可直接击穿止损线，止损在T+1下形同虚设
- **结论**：短周期策略必须基于T+1严格回测，否则结果不可信

### 教训2：数据泄漏是回测虚高的头号元凶
- 看似微小的时点偏差（如训练集截止日多含几天）可导致CAGR从真实22%虚高到160%
- 跨股票特征污染（groupby缺失）导致特征完全错误
- **结论**：任何回测必须严格Walk-Forward，任何特征必须严格逐股计算

### 教训3：交易成本是高频策略的杀手
- A股印花税（卖出0.05%）+ 佣金 + 滑点 + 冲击成本
- 每周100%换手率策略，在扣除真实成本后CAGR从正转负
- **结论**：回测必须计入0.3%以上双边成本，且需考虑流动性ADV限制

### 教训4：小盘Beta ≠ Alpha
- 等权持仓天然暴露小盘风格，大幅跑赢CSI 1000不等于有选股能力
- 必须进行风格归因（Market + SMB + Industry），验证Alpha的统计显著性（t>1.96, p<0.05）
- **结论**：未通过风格归因的"Alpha"只是Beta的伪装

### 教训5：期权特征应作为模型输入，而非硬性开关
- 硬性宏观过滤（QVIX高时暂停交易）会扼杀个股Alpha，导致收益腰斩
- 将期权特征作为XGBoost的输入特征，让模型自适应学习不同情境
- **结论**："让模型决策"优于"人工规则开关"

### 教训6：新闻/热排因子增量价值有限
- 新闻因子经风格归因后Alpha增量仅0.08%
- 同花顺热排因子是短样本噪音，加入后收益反而下降
- **结论**：热门数据不等于有效因子，需严格增量检验

### 教训7：筹码结构变化是有效信号
- 筹码Delta特征（成本分位变化、获利盘变化）重要性排名前列
- 反映了资金的真实流入/流出结构
- **结论**：筹码分布数据值得深入研究，特别是Delta变化

## 五、未来量化交易路线图

### 阶段一：立即行动（修复与验证，1-2周）

1. **严格T+1回测审计**
   - 对Study 005当前策略重新运行**严格T+1去未来函数回测**
   - 验证`backtest_options_model.py`的Worst-Case逻辑是否完整
   - 确认训练标签`d-7`提前量是否严格执行

2. **交易成本敏感性分析**
   - 在回测中纳入真实成本：印花税0.05%（卖出）+ 佣金0.025% + 滑点0.1% + 冲击成本
   - 测试不同换手率下的净收益曲线
   - 如扣除成本后CAGR为负，策略不可实盘

3. **幸存者偏差审计**
   - 使用Tushare退市股票列表，确认回测股票池是否包含已退市股
   - 若未剔除，重新运行回测

4. **PE/PB点-in-time验证**
   - 确认基本面数据是按**公告日**对齐还是财报期末日对齐
   - 后者引入严重未来函数

### 阶段二：策略迭代（1-2个月）

5. **标签重新设计**
   - 当前`+6%/-5%`标签过度拟合高波动个股
   - 尝试新标签：
     - 回归标签：直接预测T+2收盘价相对T+1开盘价的收益率（连续值）
     - 分位数标签：预测收益率在截面中的分位排名
     - 信息比率标签：预测超额收益/跟踪误差

6. **持仓周期延长实验**
   - 短周期（1-2日）受T+1约束和成本侵蚀严重
   - 测试5日、10日、20日持仓周期：
     - 降低换手率，减少成本冲击
     - 让模型有更大空间捕捉趋势Alpha
     - 与longterm-research的20日再平衡对齐

7. **特征优化**
   - 将`new_idea`中的筹码Delta特征整合到Study 005
   - 测试剔除新闻因子后的纯技术+筹码+期权模型
   - 尝试Alpha101/GTJA191的预处理后非线性建模（XGBoost/LightGBM）

8. **风险模型升级**
   - 当前仅行业中性化（单行业≤2只），缺乏Barra风格因子控制
   - 引入Size、Value、Momentum、Volatility风格暴露控制
   - 目标：让策略的纯Alpha更干净，降低风格依赖

### 阶段三：长期布局（3-6个月）

9. **多策略组合（Strategy Allocation）**
   - 单一策略无法在所有市场环境下盈利
   - 构建三层组合：
     - **底仓层**：Smart-Beta 100只小盘（长期持有，低换手）
     - **Alpha层**：20日再平衡多因子（中频，统计显著Alpha）
     - **卫星层**：短周期T+1策略（仅在高置信度时开火，低仓位）
   - 各层之间相关性低，组合平滑曲线

10. **期权增强的指数化配置**
    - 将期权Max Pain效应作为**月度权重偏置器**
    - 在临期（DTE<=3）时，根据偏离度调整ETF持仓权重
    - 不依赖做空，仅做多头端增强

11. **引入机器学习前沿方法**
    - 从XGBoost升级到**LightGBM**或**CatBoost**（处理类别特征更优）
    - 尝试**神经网络**（TabNet、FT-Transformer）挖掘非线性因子交互
    - **强化学习**（PPO/SAC）用于动态仓位管理和止损阈值自适应

12. **宏观情景识别（Regime Detection）**
    - 当前模型在牛市（2023、2025）表现好，熊市/震荡市（2024、2026）表现差
    - 引入HMM（隐马尔可夫模型）或聚类识别市场状态（牛市/熊市/震荡）
    - 不同状态下使用不同策略/不同仓位

### 阶段四：实盘纪律（持续）

13. **严格实盘审计清单**
    - 盘前：账户对账、ST过滤、次新股过滤、停牌过滤
    - 盘中：流动性ADV 1%限制、一字涨停放弃、逐Bar止盈止损
    - 盘后：交易日志、收益归因、模型漂移检测

14. **模型监控与再训练**
    - 每月Walk-Forward重训是必要但不够的
    - 监控模型预测概率的校准度（Calibration Curve）
    - 当prob_up的分布发生剧变（如2024-04月2572只prob>=0.58）时，触发人工审查

15. **资金曲线与心理状态**
    - 量化交易的敌人不是市场，是自己的预期管理
