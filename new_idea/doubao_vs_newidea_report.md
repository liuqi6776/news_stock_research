## 交易周期（修正）

```
T日（选股日）: 收盘后收集数据，模型选股
T+1日（买入日）: 开盘价买入，检查涨停，持有过夜
T+2日（卖出日）: 止盈(最高价>=买入价*(1+TP))或收盘卖出
```

## Delta特征验证

- delta = (t-1)日 - (t-2)日，预测t日
- 无未来数据泄露 ✓
