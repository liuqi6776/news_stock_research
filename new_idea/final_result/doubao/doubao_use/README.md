# doubao策略 - 每日选股完整指南

## 策略概览

doubao策略是一个**真实可执行的 T+1 量化交易策略**，完全符合 A 股交易规则，并集成了**TS综合评分过滤**。

## 完整交易规则

### 时间线（三天一个周期）

```
┌─────────────────────────────────────────────────────────┐
│  T日（第1天）- 收盘后                                     │
│  ↓                                                        │
│  1. 收集数据：热度、筹码、新闻                             │
│  2. 计算TS特征（ret_1d, delta_winner_rate等）             │
│  3. 模型选股，生成候选股                                   │
│  4. TS综合评分过滤（ts_score > 0）                        │
│  5. 保存：Top 3（概率>0.8）或 Top 1                       │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  T+1日（第2天）- 早盘                                     │
│  ↓                                                        │
│  1. 检查：是否开盘涨停？                                   │
│  2. 买入：开盘价，平均分配仓位                             │
│  3. 持有：全天持仓，隔夜                                   │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  T+2日（第3天）- 收盘前                                    │
│  ↓                                                        │
│  收盘前（14:50-14:55）：全部卖出（收盘价）                 │
└─────────────────────────────────────────────────────────┘
```

### 买入规则（T+1 日早盘）

**选股标准**：

1. **模型预测**：XGBoost 预测上涨概率
2. **主板过滤**：仅保留沪市60xxxx.SH和深市00xxxx.SZ
3. **市值过滤**：流通市值 <= 100万（circ_mv <= 1000000）
4. **首选**：概率 > 0.8 的 Top 3
5. **备选**：无概率 > 0.8 时选 Top 1
6. **TS过滤**：ts_score > 0（综合TS信号过滤）

**买入执行**：
- 买入时间：T+1 日开盘（9:30）
- 买入价格：开盘价
- 涨停检查：开盘价 >= 涨停价 → 放弃买入
- 仓位分配：选N只 → 每只 1/N 仓位

### 卖出规则（T+2 日）

- 卖出时间：收盘前（14:50-14:55）
- 卖出价格：收盘价
- 严格 T+1，不留过夜仓

### TS综合评分（ts_score）

```
ts_score = -|ret_1d| × 0.5 + delta_winner_rate × 3.0
           -|delta_chip_conc| × 1.5 - |ret_5d| × 0.2
           -|ma5_dist| × 0.3
```

| 特征 | 含义 | 权重逻辑 |
|------|------|----------|
| ret_1d | 1日涨幅 | 负权重：不追高 |
| delta_winner_rate | 获利盘变化 | 正权重：筹码改善 |
| delta_chip_conc | 筹码集中度变化 | 负权重：不稳定 |
| ret_5d | 5日累计涨幅 | 负权重：均值回归 |
| ma5_dist | 偏离5日均线 | 负权重：回归风险 |


## 模型说明

### 模型文件

- **位置**：`models/doubao_t1t2_model.joblib`
- **算法**：XGBoost 分类器
- **标签**：(T+2 close / T+1 open - 1) > 0.04
- **特征**：
  1. `hot_rank_pct` - 同花顺热度分位数
  2. `chip_concentration` - 筹码集中度
  3. `winner_rate` - 获利盘比例
  4. `news_market_impact` - 大盘新闻影响
  5. `news_stock_impact` - 个股新闻影响

### 训练参数

```python
XGBClassifier(
    n_estimators=300,
    max_depth=4,
    learning_rate=0.03,
    subsample=0.8,
    colsample_bytree=0.8,
    random_state=42,
    eval_metric='auc',
    tree_method='hist'
)
```

### 数据过滤

- 排除科创板（688/689开头）
- 流通市值 <= 100万（circ_mv <= 1000000）
- 仅主板股票（60xxxx.SH / 00xxxx.SZ）

## 注意事项

1. **首次使用**：需要先运行完整流程（1→2→3→4）
2. **日常使用**：一般只需要运行 1→2→4
3. **模型重训**：建议每月重训一次，或当策略表现明显下滑时重训
4. **数据完整性**：确保新闻数据使用 `news_major1`
5. **TS过滤**：默认开启，可通过 `--no-ts` 关闭
6. **交易成本**：回测中已包含 0.15% 的手续费
7. **涨停板检查**：已包含涨停板无法买入的逻辑
