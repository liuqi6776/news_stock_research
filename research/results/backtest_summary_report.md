## 3. 关键发现

### 3.1 阈值影响
- **阈值0.5**: 交易最频繁(171次)，但收益最差(-21.39%)，回撤最大(-53.38%)
- **阈值0.55-0.6**: 交易次数显著减少，收益接近0%，回撤控制在10%以内
- **阈值0.65+**: 几乎没有交易，策略过于保守

### 3.2 模型预测概率分布
- 平均预测概率: 13.3%
- 99%分位数: 33.5%
- 最大值: 60.6%

这表明模型整体预测概率偏低，导致高阈值下交易机会极少。

## 4. 问题诊断

### 4.1 当前策略问题
1. **预测概率偏低**: 模型输出的概率普遍较低，99%的股票概率<0.34
2. **信号稀疏**: 高阈值下交易机会极少，策略大部分时间空仓
3. **收益不稳定**: 低阈值交易频繁但亏损，高阈值过于保守

### 4.2 改进方向
1. **特征工程**: 增加更多有效特征（技术指标、基本面、市场情绪）
2. **模型优化**: 调整样本权重，解决类别不平衡问题
3. **目标函数**: 优化模型目标，提高预测概率的区分度
4. **多因子组合**: 结合多个信号源，提高信号质量

## 6. 使用说明

### 6.1 运行完整回测
```bash
python walk_forward_simple_v2.py --mode full --min_prob 0.6
```

### 6.2 调整阈值（无需重新训练）
```bash
python walk_forward_simple_v2.py --mode threshold --min_prob 0.55
```

### 6.3 批量测试阈值
```bash
python threshold_optimizer.py
```
