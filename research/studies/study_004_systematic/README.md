# Study 004: 系统化研究框架

## 研究目的
建立标准化的研究流程，自动验证多个预测周期

## 预测目标定义

| 目标 | 买入时点 | 卖出时点 | 收益计算 | 默认阈值 |
|------|----------|----------|----------|----------|
| 1日 | t+2开盘 | t+3收盘 | (t+3收盘 - t+2开盘) / t+2开盘 | 1.5% |
| 5日 | t+2开盘 | t+7收盘 | (t+7收盘 - t+2开盘) / t+2开盘 | 3% |
| 28日 | t+2开盘 | t+30收盘 | (t+30收盘 - t+2开盘) / t+2开盘 | 8% |

## 数据划分

```
2020-2021: 训练数据（最早可用数据）
2022-2024: Validation数据（用于阈值选择和网格搜索）
2025-2026: 最终测试数据（不可用于任何调参）
```

## 执行流程

1. **数据准备**：计算所有特征和三个目标变量
2. **自动阈值选择**：保证数据平衡性（正样本20%-80%）且预测有效
3. **模型训练**：用全部历史数据训练三个模型
4. **预测生成**：保存所有预测到 predictions/
5. **网格搜索**：在validation期间搜索最优止盈止损
6. **最终测试**：在2025年数据上验证（完全不可见数据）


## 结果解读

### 自动阈值选择标准
- 正样本比例在20%-80%之间
- 预测为正的样本平均收益 > 0
- 选择平均收益最高的阈值

### 网格搜索参数
- 止损: [3%, 5%, 8%, 10%]
- 止盈: [5%, 10%, 15%, 20%]
- 持仓: [3, 5, 8]只

### 最终测试
- 使用2025年数据
- 使用validation期间找到的最优参数
- 评估模型在完全不可见数据上的表现
